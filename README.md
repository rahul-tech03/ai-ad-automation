# AI Ad Creative Automation Pipeline

This project automates the creation of multiple ad variations for products using AI.

## Features
- Generates 3 ad hooks, headlines, and scripts per product
- Uses OpenAI for creative generation
- Designed for AI video tools like HeyGen
- Scales to hundreds of products using CSV input

## Workflow
1. Add products to products.csv
2. Run the Python script
3. Generate video scripts automatically
4. Render videos using HeyGen

## Tech Stack
- Python
- OpenAI API
- CSV-based input
- HeyGen (video rendering)

## Run
```bash
python main.py
